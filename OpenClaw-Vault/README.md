# OpenClaw Vault

Professional Obsidian layout for monitoring assistant state/config Markdown files.

## Folder Standard
- `00-Dashboard/` entry points and maps
- `10-Identity/` assistant identity
- `20-User/` user profile/preferences
- `30-Memory/` long-term memory
- `40-Operations/` heartbeat/ops
- `90-System/` system behavior/control files

## Notes
- Most files are symlinks to workspace source files (live view).
- Edit either from workspace root or via this vault.
